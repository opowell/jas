# JAS
A javascript app server.

# Requirements
Node.js

# Quick start
1. Download project for your platform.
2. Unzip.
3. Open terminal in /src folder.
4. `node ./server/jas.js`

# Files
- apps - files for apps
- server - files for server

# Apps
- create app folder in /apps.
- index.html required in app folder.
